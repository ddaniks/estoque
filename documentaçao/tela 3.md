# Tela de Dashboard

## Descrição
Tela de resumo do estoque do estabelecimento selecionado.

## Componentes
- **Resumo de Estoque:** Exibe uma visão geral do estoque atual.
- **Lista de Categorias:** Lista das categorias disponíveis.
- **Botão de Adicionar Categoria:** Permite adicionar uma nova categoria.
- **Botão de Adicionar Produto:** Permite adicionar um novo produto.

## Funcionalidades
- Mostrar informações atualizadas sobre o estoque.
- Navegação para telas de gerenciamento de categorias e produtos.
- Opção para trocar de estabelecimento.
